## Что это такое

Полностью кастомная LMS, созданная для Seventy Company, чтобы заменить ручной онбординг. Система охватывает структурированное обучение, экзамены с проверкой ИИ, психометрическое профилирование кандидатов и аналитику для администраторов — всё в одном продукте.

Раньше вся теория объяснялась офлайн менеджерами индивидуально для каждого нового сотрудника, без возможности надёжно проверить знания перед переходом к практике. Теперь стажёры приходят на практический этап с полным пониманием теории — и у руководства есть данные для принятия кадровых решений.

## Как это работает

1. **Новый кандидат** — получает доступ к тесту; на основе результатов автоматически формируется ИИ-портрет кандидата
2. **Профилирование** — 25 вопросов, ответы вносятся в приложении
3. **ИИ-портрет кандидата** — Gemini генерирует полный психометрический профиль: архетип, SWOT, % культурного соответствия, рекомендации по роли
4. **Доступ открыт** — если по результатам теста кандидат подходит, менеджмент открывает ему доступ к полной LMS и он становится стажёром
5. **Теория** — структурированные уроки по 3 курсам
6. **Тесты и экзамены** — открываются после прохождения теории; экзамены проверяются ИИ в реальном времени
7. **Следующий модуль** — открывается только после сдачи текущего экзамена; 24ч cooldown после провала
8. **ИИ-отчёт по обучению** — администратор на любом этапе обучения может сгенерировать полный анализ стажера: сильные стороны, слабые зоны, история экзаменов, рекомендации
9. **Решение о найме** — руководство использует портрет + отчёт для определения роли и пути развития

## Роли пользователей

| Роль | Доступ |
|---|---|
| **Кандидат** | Только тест для создания ИИ-портрета — остальное недоступно |

| **Внешний обучающийся** | Только 1 курс (Теория продукта), свои результаты, автоистечение через 7 дней |
| **Администратор** | Всё: управление контентом, сотрудниками, ИИ-отчётами, порталом кандидатов, аналитикой, редактор промптов |

## Ключевые возможности

### 📚 Структурированный процесс обучения
- 3 курса: Правила компании, Теория продукта, Рецепты
- Контент в формате историй (свайпаемые карточки); защита от копирования
- Последовательная разблокировка: следующая тема открывается только после сдачи текущего теста/экзамена
- XP-очки начисляются за каждый пройденный урок и сданную аттестацию

### 🤖 ИИ-экзаменатор (Gemini)
- Открытые вопросы на экзамене проверяются Gemini; минимум 300 символов, вставка скопированного текста запрещена
- ИИ даёт мгновенную обратную связь: балл + что не раскрыто + что улучшить
- Двуязычная поддержка: русский или казахский — ИИ определяет язык и отвечает на том же языке
- До 3 попыток на экзамен; после 3 провалов — cooldown 24 часа
- Экзамены по рецептам: интерактивный формат выбора ингредиентов и их объемов; ИИ проверяет правильность и отслеживает топ-5 слабых напитков по каждому студенту
- Полная история ответов стажера и фидбэка от ИИ сохраняется — видна студенту и администратору

### 👤 ИИ-портрет кандидата (Gemini)
- Кандидаты проходят тест из 25 вопросов
- Gemini генерирует полный психометрический отчёт:
  - Архетип (Оператор / Мастер / Ментор / Архитектор)
  - Психометрика по 10 шкалам
  - SWOT-анализ
  - % культурного соответствия ДНК компании
  - Поведенческие паттерны
  - Индивидуальные вопросы для интервью
  - Рекомендации по онбордингу
  - Подходящие роли
- Сравнение нескольких кандидатов в одном окне

### 📊 ИИ-отчёт по обучению (Gemini)
- Администратор может генерировать отчёт по каждому сотруднику на любом этапе обучения
- ИИ анализирует: пройденные уроки, точность тестов, баллы экзаменов, слабые места
- Результат: общая оценка + сильные стороны + зоны риска + рекомендации
- История отчётов сохраняется с временными метками

### 🏆 Геймификация
- Система XP, уровни, отслеживание стриков
- Лидерборд виден всем сотрудникам
- Достижения и бейджи (Энциклопедист, Спидраннер, Полиглот и др.)
- Путь к мастерству — 5 уровней прогресса по завершении курсов (Зелёное зерно → Темпер → Холдер → Мастер → Бог Эспрессо)

### 💬 ИИ HR-чатбот
- Отвечает на вопросы о правилах компании, культуре и внутренних процессах
- Работает на Gemini с кастомной базой знаний (.docx загружается через админку)
- Ответы на часто задаваемые вопросы настраиваются администратором

### 🛠 Админ-панель
- Полное управление контентом и сотрудниками
- Редактируемые ИИ-промпты по каждому модулю — все версии сохраняются после каждого изменения, с возможностью восстановить любую предыдущую
- Настраиваемая модель Gemini (Flash / Pro) или GPT для каждой ИИ-функции независимо

## ИИ-промпты

Система использует отдельные промпты Gemini для каждой ИИ-функции, все редактируются в админ-панели без изменений в коде:

- **ИИ-экзаменатор** — оценивает ответы, даёт педагогическую обратную связь на языке студента
- **ИИ-портрет** — генерирует психометрический профиль кандидата по результатам теста
- **ИИ-отчёт** — анализирует результаты обучения и формирует HR-рекомендации
- **ИИ HR-чатбот** — отвечает на вопросы о компании на основе загруженной базы знаний

См. [`prompts/ai_examiner_prompt.md`](./prompts/ai_examiner_prompt.md) — пример структуры промпта ИИ-экзаменатора.  
См. [`prompts/ai_portrait_prompt_template.md`](./prompts/ai_portrait_prompt_template.md) — пример структуры промпта ИИ-портрета.

## Влияние на бизнес

**До этой системы онбординг в Seventy Company был полностью ручным:**
- Теория объяснялась офлайн менеджерами индивидуально для каждого нового кандидата
- Не было надёжного способа проверить усвоение знаний перед переходом к практике
- Оценка потенциала сотрудника и решение о роли требовали проб и ошибок

**Теперь:**
- Стажёры приходят на практику с подтверждёнными теоретическими знаниями
- Руководство видит точно, где каждый допускал ошибки и что знает хорошо
- Решения о найме и продвижении подкреплены психометрическими данными от ИИ и ИИ-отчётами по результатам экзаменов — а не интуицией
